Projects

Super Mario Brothers: Recreated

November 2023 - December 2023

With a goal to blend nostalgia with modern web technologies, I created this project to recreate the iconic Super Mario Bros game. This initiative was driven by a passion for classic gaming experiences and a desire to demonstrate the versatility of web technologies such as HTML5, CSS3, and JavaScript. The objective was not merely to replicate the original game but to reimagine it for the web, making it accessible across devices without the need for any plugins or additional software.

Name: myKitchenManager

Spring 2023

Created a React.js web application to enhance budget management and promote balanced diets for increased future value and accessibility. Utilized the Spoonacular API to retrieve comprehensive information on various food items, including macronutrient content, pricing, serving sizes, etc. The information is stored in a Heroku and Firebase Database.
